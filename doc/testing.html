1 Testing

The test class TestAdjacencyMatrix is a JUnit test testing the following methods

  • addVertex
  • numVertices
  • insertDirectedEdge
  • inDegree/outDegree
  • outAdjacentVertices
  • removeEdge
  • reverseDirection

The test currently passes under JUnit 4. To run, import into Eclipse and run as a JUnit test.

Furthermore the tests for karate.net and c-elegans.net currently output the data produced by Dr. Suthers.
